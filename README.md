# typst-web

Generate PDFs in the browser (or on the server) using [Typst](https://github.com/typst/typst).

## Usage

See `src/App.tsx`.

## Running locally

- Install [wasm-pack](https://github.com/drager/wasm-pack).
- Run `bun run build:lib` to build the library.
- Run `bun run dev` to start the demo app.
